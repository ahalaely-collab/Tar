# خطة تطوير NexusWar 2026 - الحرب القادمة

**الهدف:** بناء نسخة حديثة (2026) من لعبة ترافيان (TravianZ) بكود برمجي قوي وآمن مع مطابقة 95%+ للميزات الأصلية.

**التقنيات المستخدمة:**
- Frontend: React 19 + TypeScript + TailwindCSS 4
- Backend: Express 4 + tRPC 11 + Node.js
- Database: MySQL 8.0 + Drizzle ORM
- Security: bcrypt, CSRF tokens, Rate Limiting, SQL Injection Prevention, XSS Protection

---

## المرحلة 1: تحليل وتصميم قاعدة البيانات

### الجداول الأساسية المطلوبة:

#### 1. جداول المستخدمين والقرى
- `users` - بيانات اللاعبين
- `villages` - القرى (كل لاعب يمكنه امتلاك عدة قرى)
- `resource_fields` - حقول الموارد (18 حقل لكل قرية)
- `buildings` - مباني القرية (22 موقع بناء)

#### 2. جداول الموارد والبناء
- `construction_queue` - طابور البناء (2 مبنى متزامن)
- `resource_production` - حساب الإنتاج الساعي
- `storage` - المستودعات والحبوب

#### 3. جداول الجيوش والحرب
- `troops` - الوحدات العسكرية
- `troop_movements` - حركات الجيوش على الخريطة
- `battles` - سجل المعارك والتقارير
- `battle_reports` - تقارير المعارك التفصيلية

#### 4. جداول الخريطة والعالم
- `world_map` - خريطة العالم (شبكة X,Y)
- `oasis` - الواحات (تعطي بونص موارد)
- `artifacts` - الآثار (تعطي قوة عسكرية)

#### 5. جداول التحالفات والاجتماعيات
- `alliances` - التحالفات
- `alliance_members` - أعضاء التحالفات
- `messages` - الرسائل بين اللاعبين
- `alliance_diplomacy` - العلاقات بين التحالفات (حرب، سلام، معاهدة)

#### 6. جداول الأبطال والمهام
- `heroes` - بيانات الأبطال
- `hero_items` - أدوات البطل
- `hero_adventures` - مغامرات البطل
- `quests` - المهام اليومية/الأسبوعية

#### 7. جداول السوق والاقتصاد
- `market_offers` - عروض السوق (بيع/شراء موارد)
- `trade_routes` - طرق التجارة بين القرى

#### 8. جداول الترتيب والإحصاءات
- `rankings` - ترتيب اللاعبين (نقاط، جيش، بناء)
- `statistics` - إحصاءات اللعبة العامة

#### 9. جداول الأمان والإدارة
- `sessions` - جلسات المستخدمين
- `admin_logs` - سجلات الإدارة
- `banned_users` - قائمة الحسابات المحظورة
- `game_config` - إعدادات اللعبة

## المرحلة 4: نظام البناء

### الميزات:
- 22 موقع بناء في القرية
- طابور بناء بـ 2 مبنى متزامن
- حساب تكاليف الترقية (حتى مستوى 20)
- تحقق من توافر الموارد
- عداد تنازلي لوقت الانتهاء
- نظام Cron لإنهاء البناء تلقائياً

## المرحلة 5: نظام الجيوش والحرب

- 6 أنواع قوات لكل قبيلة (رومان، غاليين، تيوتون، مصريين، هون، إسبارطيين)
- مباني تدريب (ثكنة، إسطبل، ورشة حصار)
- محرك معارك (Battle Engine) يحسب النتائج بناءً على:
  - قوة الهجوم والدفاع
  - نوع الوحدات
  - المباني الدفاعية
  - الأبطال والأدوات
- أنواع الحركات: هجوم، تعزيز، نهب، استكشاف
- تقارير معارك تفصيلية
